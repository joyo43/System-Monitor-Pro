Features

Real-time Monitoring

Monitor your system resources in real-time with beautiful visualizations and detailed metrics.

CPU & Memory

Track CPU usage by core, memory allocation, and historical performance patterns.

Disk & Network

Monitor disk activity, I/O operations, network interfaces, and data transfer rates.

GPU Monitoring

View GPU usage, temperature, memory, and clock speeds for supported graphics cards.

Process Management

View and manage running processes, sort by resource usage, and monitor performance over time.

Dark & Light Modes

Choose between beautiful dark and light themes to match your preference and reduce eye strain.
